# Projeto_PAC

## Descrição
Repositório criado para o projeto da disciplina de Projeto Acadêmico em Computação do curso de Ciência da computação da UFSCar, em Junho de 2018.

## Preparando o projeto
Observação: Os arquivos JSP são executados diretamente no servidor, por conta disto não é possível utilizar um caminho relativo, é necessário copiar os arquivos XML e JSON para um diretório específico rodando no servidor.
Se deseja utilizar estes arquivos para testes siga atentamente os passos:

* Abra o NetBeans e compile uma vez o projeto
* Acesse: http://localhost:8080/Projeto_PAC/caminho.jsp?
* Será retornado um endereço no corpo da página
* Copie os diretórios "json" e "xml" para dentro do endereço retornado no passo anterior
* Compile uma segunda vez o projeto e estará pronto para o uso
